
# Semantic Search Engine

- FAISS ANN
- Hybrid BM25 + Semantic
- Two-stage reranking
- FastAPI service

## Features
- Scalable vector search
- Hybrid retrieval
- Cross-encoder reranking
- Evaluation framework
